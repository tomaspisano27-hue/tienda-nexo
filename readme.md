Quiero que actúes como un diseñador UX/UI senior, desarrollador Full Stack y especialista en ecommerce B2B.

Desarrollá una página web completa para mi negocio mayorista llamado **Tienda Nexo**.

## Sobre el negocio

* Nombre: **Tienda Nexo**
* Rubro: Distribuidora mayorista de productos para kioscos.
* Vendemos **productos NO comestibles**.
* Comercializamos a comercios, distribuidores y revendedores de **todo Argentina**.
